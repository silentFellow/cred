## cred ssh cp

copies files and directories

### Synopsis

The cp command allows you to copies files and directories.
It uses the 'cp' command to move the specified file or directory to a new location.

Examples:
  cred ssh cp <source> <destination>

```
cred ssh cp [flags]
```
