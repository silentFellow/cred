## cred env rm

Remove files and directories

### Synopsis

The rm command allows you to remove files and directories recursively.
It uses the 'rm' command to delete the specified path.

Examples:
  cred env rm <path>

```
cred env rm [flags]
```

### Options

```
  -h, --help   help for rm
```
