## cred ssh show

Displays the stored ssh

### Synopsis

The show command retrieves and displays the stored ssh key for a given file.
Usage examples:

cred ssh show <filename>

```
cred ssh show [flags]
```

### Options

```
  -h, --help   help for show
```
