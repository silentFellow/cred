## cred ssh mv

Move files and directories

### Synopsis

The mv command allows you to move files and directories.
It uses the 'mv' command to move the specified file or directory to a new location.

Examples:
  cred ssh mv <source> <destination>

```
cred ssh mv [flags]
```
