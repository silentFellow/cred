## cred env ls

List files and directories

### Synopsis

The ls command allows you to list files and directories.
It uses the 'ls' command to display the contents of the current directory.

Examples:
  cred env ls <path>

```
cred env ls [flags]
```

### Options

```
  -h, --help   help for ls
```
