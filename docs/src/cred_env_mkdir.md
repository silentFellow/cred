## cred env mkdir

Create directories

### Synopsis

The mkdir command allows you to create directories, including nested directories.
It uses the 'mkdir' command to create the specified path.

Examples:
  cred env mkdir <path>

```
cred env mkdir [flags]
```

### Options

```
  -h, --help   help for mkdir
```
